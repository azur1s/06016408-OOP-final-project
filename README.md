# Object-Oriented Programming (2/68) Final Project
[Thai Version](./README_th.md)

To see the design details and architecture, see [ARCHITECTURE.md](./ARCHITECTURE.md).

## Compiling

### 1. Prerequisites
- Java 17 or higher
- Maven (See instruction from the [Maven Official Guide](https://maven.apache.org/install.html))

### 2. Open Terminal
```sh
# Go to project folder
cd project

mvn compile exec:java
# MacOS version
mvn compile exec:exec
```
